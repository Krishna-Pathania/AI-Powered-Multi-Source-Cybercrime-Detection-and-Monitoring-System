# 🛡️ AI-Powered Cybercrime Detection System

A real-time multi-source cybercrime detection system that identifies phishing URLs, scam messages, and suspicious activity using Machine Learning, NLP, and a monitoring dashboard.

---

## 🚀 Features

- 🔍 Phishing URL Detection using ML models
- 🧠 Scam Message Detection using NLP (TF-IDF + Logistic Regression)
- 🌐 Browser Extension for automatic URL scanning
- 📋 Background Clipboard Monitoring Agent
- 📊 Streamlit Dashboard with:
  - Live Alerts
  - Scan History
  - Detailed Analysis
- ⚠️ Real-time alert system
- 📉 Risk scoring with explainable AI

---

## 🧠 Technologies Used

- Python
- Flask (Backend API)
- Streamlit (Dashboard)
- Scikit-learn (ML models)
- TF-IDF (NLP processing)
- JavaScript (Browser Extension)
